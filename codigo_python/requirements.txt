# Calculadora de Alíquota do ISS - VBA

## Instruções de Uso

1. Abra o Excel e acesse o Editor do VBA (Alt + F11).
2. Importe o script `calculator.vba`.
3. Adicione os dados na planilha conforme o exemplo.
4. Execute a macro `CalcularAliquota`.

## Estrutura do Código

- `calculator.vba`: Contém a lógica para calcular a alíquota do ISS no Excel.
